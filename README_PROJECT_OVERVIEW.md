# AI Language Coach: Proyecto de Guerrilla 🚀

## Misión
Una plataforma de aprendizaje de inglés minimalista, sin distracciones y potenciada por Inteligencia Artificial generativa (Gemini). Diseñada para llevar el conocimiento teórico (palabras sueltas) a la práctica real (escenarios de conversación).

## Características Principales
- **AI Scenario (Roleplay Dinámico)**: Practica en situaciones reales con un tutor IA que analiza tu gramática y sugiere formas naturales de expresión.
- **Word Matcher**: Juego de alta velocidad para emparejar términos y calentar motores.
- **The Knowledge Vault (El Baúl)**: Almacén de frases y palabras dominadas que sirven de contexto para personalizar las sesiones de IA.
- **Raccoon Coach**: Tu guía personal que proporciona feedback amigable.

## Filosofía de Desarrollo: "Zero-Dependency Minimalist"
- **Sin Dependencias Externas**: Todo se construye con componentes core de React Native (`View`, `Text`, `Animated`, `TouchableOpacity`, `StyleSheet`).
- **Contrato JSON**: Comunicación estructurada con la IA para renderizar feedback elegante.
- **UI Limpia**: Espacio en blanco, tipografía clara y navegación intuitiva.

## Guía de Inicio Rápido
1. Instalar dependencias: `npm install`
2. Iniciar proyecto: `npx expo start`
